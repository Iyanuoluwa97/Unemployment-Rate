# R Programming: Data Operations & Visualisation

This directory contains a script containing R programming inputs and commands for statistical computation and data plotting.

---

## Script Overview

The script demonstrates foundational R operations, vector math, and visual plotting. It executes calculations on sequences and datasets and visualizes them using R's built-in plotting engines.

### Key Operations Implemented:
- **Sequence Generation**: Creating sequences of numbers utilizing R's `seq()` function.
- **Statistical Calculation**:
  - Calculating arithmetic mean using the `mean()` function.
  - Determining vector lengths via the `length()` function.
  - Computing statistical deviations: calculating sum of squared differences from the mean normalized by degree of freedom (`sum(x-mean(x))^2/length(x)-1`).
- **Data Plotting**: Generating custom bar charts (`barplot`) with custom axes labels (`xlab`, `ylab`), titles (`main`), color vectors (`skyblue`), and custom x-axis category labels.
